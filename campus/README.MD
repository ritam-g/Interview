# 2. What Is OOP?

### Question

What is OOP?

### Answer

> OOP stands for Object-Oriented Programming.
>
> It is a programming approach where we organize code using classes and objects.
>
> OOP helps make code reusable, manageable, and easier to maintain.

### Real-Life Example

Car is a class.

My Honda City is an object.

Many cars can be created from one class.

### Answer

# 18. Difference Between var, let, and const

### Question

Difference between var, let, and const?

### Answer

| var            | let              | const            |
| -------------- | ---------------- | ---------------- |
| Can redeclare  | Cannot redeclare | Cannot redeclare |
| Function scope | Block scope      | Block scope      |
| Can reassign   | Can reassign     | Cannot reassign  |

### Interview Answer

> var is older.
>
> let is used when value can change.
>
> const is used when value should not change.
